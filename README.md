# Ejercicio GitHub: Entornos de Desarrollo

## Descripción
Este repositorio contiene el ejercicio práctico sobre **entornos de desarrollo** utilizando Git y GitHub. El objetivo del ejercicio es familiarizarse con la creación, configuración y manejo de proyectos en diferentes entornos, así como con el uso básico de control de versiones.

---

## Objetivos del ejercicio
1. Comprender el conceptos básico de GIt.  
2. Aprender a configurar proyectos en GitHub.  
3. Practicar el uso de ramas, commits y pull requests.  
4. Colaborar en proyectos mediante control de versiones.  

---

## Requisitos previos
- Tener instalado **Git**.  
- Tener una cuenta en **GitHub**.  
- Conocimientos de línea de comandos.
